# bodyclock.fm

Personal website and blog for Patrick Perkins.

Built with [Astro](https://astro.build).

## Development

```bash
npm install
npm run dev
```

## Writing

Create a new `.md` file in `src/content/blog/`:

```markdown
---
title: "Post title"
date: 2026-06-14
tags: ['personal', 'tech']
description: "Optional description"
---

Your content here.
```
